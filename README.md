# Community School Homepage

A static, responsive homepage for a Community School. Designed with clean semantic HTML5 structures and styled with custom vanilla CSS3.

## Features

- **Responsive Grid Layouts**: Clean stats cards and program descriptions that adjust to mobile viewports.
- **Academic Programs catalog**: Detail pages/cards for Early Education, Primary School, and Middle School.
- **Core Values section**: Features detailing the school's educational philosophy and vision.
- **School Calendar**: Upcoming events (Open Houses, sports matches, science exhibitions) presented in a clear timeline.
- **Warm color palette**: Styled using a premium forest green and cream/gold color scheme with custom CSS variables.

## Files

- [index.html](index.html): Document containing semantic layout and page content.
- [styles.css](styles.css): Stylesheet containing custom variables, typography controls, and layouts.

## How to Run

Simply open the `index.html` file in any modern web browser.
